## 💻 コードブロック

### インラインコード

これは `print("Hello, World!")` のようなインラインコードです。

### 複数行コード（Python）

```
def greet(name):
    print(f"Hello, {name}!")

greet("World")
```

### 複数行コード（指定付き）

```python
import math

def area_of_circle(radius):
    return math.pi * radius ** 2

print(area_of_circle(5))
```
